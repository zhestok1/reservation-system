# Matrix Calculator 🧮

Веб-приложение для выполнения операций над матрицами с удобным REST API.

## 📚 Документация API

Полная документация всех методов и эндпоинтов доступна по адресу:
👉 [**matrix-calculator JavaDoc**](https://zhestok1.github.io/matrix-calculator/)

## 🚀 Функционал

Калькулятор поддерживает следующие операции с матрицами:

| Операция | Описание |
|----------|----------|
| **➕ Сложение** | Сложение двух матриц одинаковой размерности |
| **➖ Вычитание** | Вычитание одной матрицы из другой |
| **✖️ Умножение** | Умножение двух матриц (с учётом размерностей) |
| **🔄 Транспонирование** | Перестановка строк и столбцов матрицы |
| **📊 След матрицы** | Сумма элементов на главной диагонали |
| **🔢 Определитель** | Вычисление детерминанта квадратной матрицы |
| **↩️ Обратная матрица** | Нахождение обратной матрицы (для невырожденных матриц) |
